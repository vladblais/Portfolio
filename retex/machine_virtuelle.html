Machine virtuelle

Cahier des charges

Mettre en place un poste de travail sur une machine virtuelle pour un développeur. La machine virtuelle doit embarquer une bureau léger de type XFCE et un IDE pour pouvoir coder dans le langage GO. La machine doit faire moins de 10 GO.

Outils utilisés

  • VMware
  • Bash
  • Fedora

Compétences acquises

  • Manipulation de machines virtuelles
  • Administration de systèmes
  • Installation de poste
  • Utilisation de Bash
  • Lire de la documentation utile en ligne
